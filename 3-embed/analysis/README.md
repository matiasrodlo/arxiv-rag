# Embedding Analysis

Tools for analyzing chunk quality and embedding readiness.

## Scripts

- **`analyze_for_embeddings.py`** - Comprehensive analysis for embedding model compatibility
- **`final_embedding_analysis.py`** - Final validation of optimized chunks

## Usage

```bash
python analyze_for_embeddings.py <chunks_dir>
python final_embedding_analysis.py <chunks_dir>
```

## Features

- Token count analysis
- Text quality checks
- Context validation
- Embedding-specific issue detection
